# uLEAF

![uLEAF Development Board](./uleaf.png)

**uLEAF** is a minimalistic STM32 platform for developing audio synthesis applications. It was primarily designed for use with [LEAF (Lightweight Embedded Audio Framework)](https://github.com/spiricom/LEAF), but is not limited to it. 

### Hardware Features
* **Powerful MCU:** Cortex-M33 (STM32H533) with memory-mapped 8 MB PSRAM
* **Storage:** 8 MB additional external Flash memory
* **Audio:** Audio DAC PCM5102A

### Design Philosophy
This development board tries to be as cheap and simple as possible. It is designed to be highly versatile, allowing you to easily embed it directly into your own projects or use it as a starting point to develop your own custom PCB based on this design.
